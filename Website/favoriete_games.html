Favoriete games

Op deze pagina ga ik het hebben over mijn favoriete games die ik vaak speel. Ik ga het o.a hebben over krunker, wolfstein 3d en War Brokers.

Krunker is een van mijn favoriete games. Het is de meest populaire free online multiplayer shooter game waar je tegen andere spelers speelt. Krunker is ontwikkeld door Yedis Entertainment en FRVR. Krunker is voor het eerst uitgegeven op 20 mei, 2018. 8 players kunnen een game joinen en tegen elkaar spelen voor een objective. De game kan je op Windows PC, Mac, ChromeOS en Linux spelen.

Link naar Krunker: Krunker.io

Wolfstein 3D is een game waar ik veel van hou. Wolfstein 3D is een first person shooter ontwikkeld door en gepubliceerd door Apogee software en FormGen. Wolfstein 3D is voor het eerst uitgegeven op 5 mei, 1992. In de game speel je tegen nazi soldiers, honden en andere karakters met messen en wapens.



Link naar Wolfstein 3D: Wolfstein 3D

War Brokers is een game die ik heel vaak speel. War brokers is ook een first person shooter spel ontwikkeld door Trebuchet Entertainment LLC. War Brokers heeft vier verschillende game modes:

  • Battle Royale
  • Classic 8v8
  • Speed 4v4
  • Death Match Royale
